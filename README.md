#scripts

1. install `pnpm` and update dependencies

```bash
npm i -g pnpm
pnpm i
```

2. development mode: `dev`

```bash
pnpm run dev
```

3. build a bundle, (including css): `build`

```bash
pnpm run build
```

4. preview
   usage example in /staging

```bash
pnpm run preview
```

and then open your browser at localhost:3000/staging.html
